Personalised JPEG Quantization Table

Part of the JPEG image format standard includes a "Quantization Table" (an 8*8 matrix/array of values), this essentially helps compress an image by going over each 8*8 block in a given image with it. Below is the table often used based on psycho-visual experiments, but by using the sliders next to the table, you can manipulate it to compress the image how you deem fit, then click the Quantize button to see the results.

A few additional features added are:

  • Upload your own photo to see how compression will work on it.
  • Copy your customised quantization table (into a format most programming languages can use)
  • Click on part of the original (before) image, to see how the compression would effect that segment of the image.
